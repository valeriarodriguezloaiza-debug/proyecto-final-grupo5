1. Home (Inicio)

¡Bienvenidos a la documentación oficial del Grupo 5! En este sitio web queremos mostrarles paso a paso cómo diseñamos y construimos desde cero una infraestructura Cloud local para el proyecto final de Sistemas Operativos.

El reto principal que asumimos como equipo fue actuar como auténticos Arquitectos Cloud. Tuvimos que enfrentarnos a un ecosistema mixto muy interesante: por un lado, configuramos un entorno gráfico usando Pop!_OS 22.04 LTS para gestionar las herramientas principales, y por el otro, levantamos un servidor puro de consola con Rocky Linux 9.4, logrando que ambas máquinas se comunicaran de forma totalmente segura a través de SSH.

Pero no nos quedamos solo en la virtualización tradicional. Para hacer el despliegue moderno y eficiente, aislamos los servicios de Frontend (Nginx) y Backend (Python) usando contenedores independientes creados sobre una imagen base de rockylinux:9 y los conectamos usando Docker Compose. Finalmente, llevamos la aplicación a producción emulando un entorno corporativo de alta disponibilidad mediante un clúster local de Kubernetes con Minikube, donde logramos escalar nuestro despliegue a 3 réplicas en vivo.

A lo largo de esta página encontrarán todas nuestras evidencias, las capturas de pantalla con los comandos reales ejecutados en consola, los diagramas de arquitectura que diseñamos y las lecciones que aprendimos rompiéndonos la cabeza con las configuraciones. ¡Esperamos que disfruten el recorrido técnico!
